# 04 中文 IR、中文汇编与中文字节码

## 分层目标

本项目的中文化不是只停留在源码关键字，而是逐步向下扩展：

```text
中文源码
中文错误信息
中文 AST 打印
中文 IR
中文汇编助记符
中文字节码
真实 RISC-V 机器码
```

## 中文 IR

普通三地址码：

```text
t0 = a + b
return t0
```

中文打印形式：

```text
临0 = 加 甲, 乙
返 临0
```

函数 IR：

```text
函 @加(甲: 整, 乙: 整) -> 整 {
入口:
    临0 = 加 甲, 乙
    返 临0
}
```

内部实现可以仍然使用枚举和结构体，中文只是 IR 的文本格式和调试格式。

## 中文汇编

RISC-V 原始汇编：

```asm
li a0, 7
ret
```

中文汇编表示：

```text
载即 参0, 7
返
```

寄存器也可以中文化：

| 中文 | RISC-V |
|---|---|
| 零 | x0 |
| 返址 | ra |
| 栈 | sp |
| 帧 | s0/fp |
| 参0 | a0 |
| 参1 | a1 |
| 临0 | t0 |
| 临1 | t1 |

第一版不需要真正实现中文汇编器。可以先让后端输出标准 RISC-V 汇编。等后端稳定后，再增加一个中文汇编打印器。

## 中文字节码

中文字节码适合做 VM 实验：

| 指令 | 含义 |
|---|---|
| 入常 | push constant |
| 加 | add |
| 减 | sub |
| 乘 | multiply |
| 除 | divide |
| 载局 | load local |
| 存局 | store local |
| 跳 | jump |
| 若零跳 | branch if zero |
| 调 | call |
| 返 | return |

例如：

```han
整 主() {
    返 1 + 2 * 3;
}
```

可以编译成：

```text
入常 1
入常 2
入常 3
乘
加
返
```

这就是“中文机器码表示”的软件实验版本。

## 与真实机器码的关系

真实 CPU 不执行汉字，而执行比特。

因此最终链路是：

```text
中文源码
  -> 中文 IR
  -> 中文汇编 / 标准 RISC-V 汇编
  -> RISC-V 机器码
```

或者：

```text
中文源码
  -> 中文字节码
  -> han-vm 解释执行
```

两条路线可以并行发展。
